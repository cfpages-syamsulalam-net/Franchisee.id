## Security Decision

The user explicitly requested D1 storage instead of environment secrets. Credentials are therefore application-readable plaintext in D1. Dashboard reads expose only `has_api_key` / `has_api_secret`; secret values are never returned to the browser or included in audit payloads. Envelope encryption with a separately held root key remains the recommended production hardening step.
