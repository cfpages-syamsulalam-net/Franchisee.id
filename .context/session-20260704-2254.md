## Findings
- Core code path exists for login/register, free franchisor/franchisee completion, profile access, Premium order creation, payment confirmation, and admin approval.
- Site is not ready for significant paid traffic yet because homepage brand/CTA hierarchy, `/profil` next-step prominence, and production QA still need work.
- `/profil` currently feels too white/light-dashboard compared with the homepage/Premium visual language; it should use stronger yellow/black/warm site tokens and a clear next-best-action panel.
- Root homepage has visible `Franchise.id` brand/copy while app pages use `Franchisee.id`, which should be resolved before acquisition traffic.
