## D1 Public Page Generation Added
Built `scripts/build-d1-franchise-pages.ts` as the first D1-backed public listing/detail generator.

Purpose:
- Generate SEO-indexable static HTML from D1 before the full Astro scaffold exists.
- Use existing WordPress-exported templates as the rendering surface.
- Keep the implementation ready to port into Astro static generation later.

Commands:

```bash
pnpm run build:d1:franchises:dry
pnpm run build:d1:franchises
```

Implementation notes:
- Queries published `site_franchisee_id` rows from `franchise_db`.
- Reads the `franchise-network` token from the local cfman token store, then calls Wrangler directly. This avoids cfman's SQL argument splitting behavior for long SELECT commands.
- Renders `/peluang-usaha/index.html`.
- Renders `/peluang-usaha/{slug}/index.html` with a `d1-generated:franchisee.id` marker.
- Regenerates `json/unclaimed-brands.json` from D1 unclaimed rows.
- Writes `json/d1-generated-pages-manifest.json` with page hashes and D1-owned paths.
- Prunes only pages tracked in the manifest and containing the D1 marker; untracked legacy/example pages are intentionally preserved.
